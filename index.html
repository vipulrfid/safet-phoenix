About this project · transparency

ML model
Decision Tree (max_depth=12, min_samples_leaf=5)
Training data
624,147 raw incidents → 26,650 aggregated cells
Aggregation
ZIP × day-of-week × 6-bucket time-of-day × month
Coverage
50 Phoenix-proper ZIPs (of 119 in the feed)
Time period
2015-11-01 to 2025-09-02 (~10 years)
Cross-validation accuracy
0.839 ± 0.005 (5-fold, shuffled)
Risk tiers
Tertile bucket of annualized incident rate · Low/Moderate/High
Confidence
Based on incident count per cell over 10 years
Crime categories
Violent · property · other (FBI/UCR groupings)
NLU layer
OpenAI gpt-4o-mini via Cloudflare Worker (rules parser fallback offline)
Source
phoenixopendata.com — Phoenix PD crime data

Disclaimer. SafeT predicts a relative risk tier (Low / Moderate / High) for a given location and time, based on historical Phoenix PD crime patterns. It does not forecast specific incidents, and predictions are not real-time — the underlying data is refreshed periodically, not live. The model excludes calls outside Phoenix PD's jurisdiction (Scottsdale, Tempe, Mesa, Chandler, Gilbert, Glendale, Peoria, etc. publish their own feeds separately). Do not use as the sole input for safety decisions.
